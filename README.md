🤖 AI QA Bot: Автоматизована система контролю якості дзвінків
Цей бекенд-сервіс на базі Telegram-бота (Aiogram) автоматизує прослуховування та оцінювання діалогів менеджерів із клієнтами. Проєкт інтегровано з Google Drive (робота з аудіо та генерація звітів), Google Sheets (ведання логів) та Google Gemini AI (транскрибація та аналіз).

✨ Ключові фічі (Features)
Повний цикл обробки: Автоматичний пошук аудіо (.m4a, .mp3), аналіз ШІ, створення текстового розбору (.txt) та переміщення оброблених файлів у цільову папку Drive.

Edge Cases (Технічний брак): Система розпізнає поширену проблему АТС (односторонній запис, де відсутній голос менеджера), блокує нарахування балів та автоматично маркує такі рядки в таблиці червоним кольором.

OAuth 2.0 Auth: Використання Desktop Application Flow для повноцінного доступу до Google Workspace користувача без обмежень сервісних акаунтів.

Структурована аналітика: ШІ (Gemini 2.5 Flash) оцінює дзвінок за 4 суворими бізнес-критеріями з поверненням валідного JSON.

🛠 Інструкція для локального запуску через IDE (PyCharm / VS Code)
Цей проєкт налаштований для швидкого розгортання в будь-якому сучасному Python IDE.

1. Підготовка проєкту
Склонуйте репозиторій та відкрийте папку проєкту у вашому IDE.

Bash
git clone https://github.com/fulejune-ops/autobot_test.git
Встановіть залежності через термінал IDE (переконайтеся, що активовано віртуальне середовище):

Bash
pip install -r requirements.txt
2. Конфігурація оточення
Скопіюйте файл .env.example та перейменуйте його на .env.

Вставте ваші приватні токени та ID папок у готові змінні (ідентифікатор Google Таблиці вже вшито за замовчуванням).

3. Авторизація Google API
Для роботи з Google Workspace покладіть ваш файл credentials.json (OAuth client ID для Desktop app, отриманий у Google Cloud Console) у кореневу папку проєкту.

4. Запуск 🚀
Відкрийте файл main.py та натисніть кнопку Run (▶️) у вашому IDE.

Примітка: При першому запуску відкриється вікно браузера для підтвердження доступу Google-акаунта. Після цього згенерується файл token.json, і бот працюватиме автономно.
